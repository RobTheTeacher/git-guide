How to change to another branch

Move to a different branch: git checkout branch-name

Move to a remote branch: git branch -r (to see remote branches) git checkout branch-name

Branch Commands

git branch: shows current branch

git branch -v: shows verbose information on branches

git branch -r: shows remote branches

git branch -a: shows all branches

How to create a branch

In order to create a branch you type:

git checkout -b branchname
