Branch Commands

Show current branch: git branch

Show verbose information on branches: git branch -v

Show remote branches: git branch -r

Show all branches: git branch -a

How to create a branch

Create a new branch: git checkout -b branchname

How to change to another branch

Move to a different branch: git checkout branch-name

Move to a remote branch: git branch -r (to see remote branches) git checkout branch-name

How to push a new branch

Type git push

Then this will appear: git push --set-upstream origin branchname

Copy and paste this in the terminal
