Why is the stash command useful?

You can use the git stash command when you want to record the current state of the working directory and the index, but want to go back to a clean working directory. The command saves your local modifications away and reverts the working directory to match the HEAD commit.

Detailing Git Stash command.

Use git stash when you want to save your local content/modifications, but go back to a clean working directory.
The commands you 'stashed away' will be showed to you when using git stash list.
Inspect with git stash show and restore with git stash apply.
Calling git stash is equivalent to git stash push.

Git Stash Pop

Git stash pop applies a removed single stashed state from the stash list, on top of the current working state tree.
The working directory have to match the index.
