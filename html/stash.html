Detailing Git Stash command.

Use git stash when you want to save your local content/modifications, but go back to a clean working directory.
The commands you 'stashed away' will be showed to you when using git stash list.
Inspect with git stash show and restore with git stash apply.
Calling git stash is equivalent to git stash push.
